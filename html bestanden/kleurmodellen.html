Kleurmodellen

Wat zijn kleurmodellen ?

Kleurmodellen zijn cijfers die je opschrijft als code. De code die je hebt opgeschreven is een soortnaam voor de kleur. Bijv. je schrijft door middel van de RGB model een code voor de achtergrond van je website. Je wilt een groene achtergrond, wat je doet is dat je dan (0,255,0) schrijft. Het programma weet nu dat je een code hebt gemaakt en die code zegt dat je een groene achtergrond wilt. De code voert dat uit en dan heb je een groene achtergrond op je website. Ze worden niet alleen gebruikt voor achtergronden van websites maar ook in tv’s, laptop schermen en smartphone schermen.

RGB

RGB staat voor Red Green Blue in het Nederlands betekent Rood Groen Blauw. Het RGB model wordt meestal gebruikt voor digitale schermen zoals tv’s, laptops en telefoon schermen. Het model bestaat uit drie kleuren: Red Green en Blue, als de kleuren samen mixt kan je alle kleuren maken. Ze hebben allemaal wel een waarde van 0 t/m 255. Bijvoorbeeld je wilt een rode kleur maken dan doe je (255,0,0) of je wilt een blauwe kleur (0,0,255). Bij zwart zet je alle getallen op nul (0,0,0) en bij wit zet je juist alle getallen op 255 (255,255,255). Als je een screenshot van een afbeelding zou maken en dan zou inzoomen dan ga je merken dat de afbeelding vager wordt en daarna nog meer zou inzoomen, dan ga je allemaal vierkantjes zien die vierkantjes worden pixels genoemd. Als je nog verder kon inzoomen dan zou je zien dat één pixel bestaat uit drie lampjes rood, groen en blauw. Je kan de lampjes harder en zachter aanzetten, zo worden de kleuren ook gevormd. Tussen de resolutie en de pixels zit een verband. Het verband tussen die twee is hoe meer pixels er in een plaatje hoe scherper het plaatje is.

CMYK

CMYK staat voor Cyan, Magenta, Yellow en Key Black in het Nederlands betekent dat Cyaan, Magenta, Geel en zwart. Het CMYK model wordt juist weer gebruikt voor als je iets wilt uitprinten. Het worden ook wel de uitprint kleuren genoemd. De reden dat dit model wordt gebruikt als je iets gaat uitprinten is omdat je RGB model gebaseerd is op de zwarte achtergrond van je scherm. Maar het papier waarop je iets gaat uitprinten is juist weer wit. Dan heb je dus een probleem, want de kleuren gaan er anders uitzien. Daarom is de CMYK model heel erg handig zodat je de kleuren kan uitprinten die je op het plaatje ziet. Bijv. je wilt een afbeelding uitprinten op scherm is gebaseerd op het RGB model en als je dat met dit model uitprint krijgt je andere kleuren te zien op het uitgeprinte plaatje en wat er op je scherm staat. Om dat te voorkomen wordt het CMYK model gebruikt zodat de kleuren van het plaatje die uitgeprint is overeenkomen met de kleuren van het plaatje die je op je scherm ziet. Als je alleen CMY hebt en dat samen mengt krijg je grijs en om toch een zwarte kleur te krijgen hebben ze Key Black toegevoegd, omdat je dan toch de zwarte kleur kan uitprinten. Anders zou je alleen een grijze kleur hebben en geen zwarte.
