kleurmodellen

Wat zijn kleurmodellen ? Kleurmodellen zijn cijfers die je opschrijft als code. De code die je hebt opgeschreven is een soortnaam voor de kleur. Bijv. je schrijft door middel van de RGB model een code voor de achtergrond van je website. Je wilt een groene achtergrond, wat je doet is dat je dan (0,255,0) schrijft. Het programma weet nu dat je een code hebt gemaakt en die code zegt dat je een groene achtergrond wilt. De code voert dat uit en dan heb je een groene achtergrond op je website. Ze worden niet alleen gebruikt voor achtergronden van websites maar ook in tv’s, laptop schermen en smartphone schermen. RGB RGB staat voor Red Green Blue in het Nederlands betekent dat Rood Groen Blauw. Het RGB model wordt meestal gebruikt voor digitale schermen zoals tv’s, laptops en telefoon schermen. Het model bestaat uit drie kleuren: Red Green en Blue, als de kleuren samen mixt kan je alle kleuren maken. Ze hebben allemaal wel een waarde van 0 t/m 255. Bijvoorbeeld je wilt een rode kleur maken dan doe je (255,0,0) of je wilt een blauwe kleur (0,0,255). Bij zwart zet je alle getallen op nul (0,0,0) en bij wit zet je juist alle getallen op 255 (255,255,255). Als je een screenshot van een afbeelding zou maken en dan zou inzoomen dan ga je merken dat de afbeelding vager wordt en daarna nog meer zou inzoomen, dan ga je allemaal vierkantjes zien die vierkantjes worden pixels genoemd. Als je nog verder kon inzoomen dan zou je zien dat één pixel bestaat uit drie lampjes rood, groen en blauw. Je kan de lampjes harder en zachter aanzetten, zo worden de kleuren ook gevormd.
