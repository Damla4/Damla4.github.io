kleurmodellen

Wat zijn kleurmodellen ? Kleurmodellen zijn cijfers die je opschrijft als code. De code die je hebt opgeschreven is een soortnaam voor de kleur. Bijv. je schrijft door middel van de RGB model een code voor de achtergrond van je website. Je wilt een groene achtergrond, wat je doet is dat je dan (0,255,0) schrijft. Het programma weet nu dat je een code hebt gemaakt en die code zegt dat je een groene achtergrond wilt. De code voert dat uit en dan heb je een groene achtergrond op je website.
